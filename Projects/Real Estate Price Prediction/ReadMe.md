# 🏠 Real Estate Price Prediction (Machine Learning)

## 📌 وصف المشروع (Description)
نظام ذكي يعتمد على تعلم الآلة (**Machine Learning**) للتنبؤ بأسعار العقارات بدقة عالية بناءً على مجموعة من البيانات التاريخية. يهدف المشروع إلى توفير تقديرات دقيقة للأسعار بناءً على ميزات مثل المساحة، الموقع، عدد الغرف، ودخل السكان في المنطقة.

## 🚀 النتائج والتقييم (Results)
تم استخدام نموذج **RandomForestRegressor**، وحقق النتائج الاحترافية التالية:
* **📊 دقة النموذج (R2 Score):** 0.8824 ($88.24\%$) - دقة قوية جداً.
* **📉 متوسط الخطأ (RMSE):** 120,271.08 (ومتوسط الخطأ في التوقعات).

[Visual Image](results_visualization.png) ## 🛠️ الأدوات المستخدمة (Tech Stack)
* **Language:** Python
* **Library:** Pandas, NumPy, Scikit-Learn
* **Visualization:** Matplotlib, Seaborn

## 💻 طريقة التشغيل (Usage)
